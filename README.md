## Run Locally

Clone the project

```bash
  git clone https://github.com/yourusername/eventribe.git
```

Go to the project directory

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server ./frontend

```bash
  npm run start
```

Start backend 

```bash
  node server.js
```
